Frontend and server functions for recipebuddy.us.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Requires a running postgres server with .env populated with the connection string.

Required .env values:
AUTH0_SECRET
APP_BASE_URL
AUTH0_DOMAIN
AUTH0_CLIENT_ID
AUTH0_CLIENT_SECRET
APP_URL
NEXT_PUBLIC_BASE_URL
ENV
AUTH_SECRET
AUTH_GOOGLE_ID
AUTH_GOOGLE_SECRET
POSTGRES_URL
POSTGRES_PRISMA_URL
POSTGRES_URL_NO_SSL
POSTGRES_URL_NON_POOLING
POSTGRES_USER
POSTGRES_HOST
POSTGRES_PASSWORD
POSTGRES_DATABASE
OPENAI_API_KEY
DATABASE_HOST
DATABASE_NAME
DATABASE_USER
DATABASE_PASSWORD
GOOGLE_GENERATIVE_AI_API_KEY
AUTH_SECRET
BLOB_READ_WRITE_TOKEN
REDIS_URL

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
